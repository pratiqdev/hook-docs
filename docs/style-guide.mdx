  reset

<div>{value}</div>

## Config

Values used for initial state and internal settings are provided as a config
object 


```ts
// config object
{
  config: 'values',
  used: ['for', 'FOR'],
  initial: 'state',
  internal: 'settings',
  or: 'whatever'
}

// interface
{
  config: string;
  used?: string[];
  initial?: string;
  internal?: number;
  or?: SomeInterface | AnotherInterface;  
}
```

| Key | Type | Default | Description |
|:--|:--|:--|:--|
`config` | string | 'beep-bop' | A fake config option for stype-guide
`values` | string[] | [] | Another placeholder config variable
