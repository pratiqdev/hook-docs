# useArray
Hook for managing the state of an array with common manipulation methods.


<!--=========================================================================-->
<hr />

## Usage

<Tabs>
<TabItem value='simple' label='Simple'>

```js 
import { useArray } from '@pratiq/hooks'

const pkg = useArray([])

pkg.set([1,2,3])

<div>{pkg.value}</div>
```

</TabItem>
<TabItem value='advanced' label='Advanced'>

```js
import { useArray } from '@pratiq/hooks'

const initialState = [1,2,3]

const {
  value, 
  set, 
  push, 
  pop, 
  shift, 
  unshift,
  reset
} = useArray(initialState)
```

</TabItem>
</Tabs>

## Hook Methods

State modification methods privided by `useArray`



### set
Set the state to a new array. Value must be of type array.

```ts
set: (v:any[]) => void

const { value, set } = useArray([])
set([1,2,3])
set([{text: "Hello, "}, {text: "World"}])
```
<br />


### clear
Set the state to an empty array.

```ts
() => void

const { value, clear } = useArray([])
clear()
```

<br/>


### reset
ResetArray the array to the initial value.

```ts
() => void 

const { value, reset } = useArray([])
reset()
```

<br/>



### push
Push a new element into the end of the array.

```ts
(...e: any) => void

const { value, push } = useArray([])
push(7)
```




<br />

### unshift
Adds new elements to the beginning of an array, and returns the new length

```ts
(...values:any) => void 

const { value, sort } = useArray([])
sort((a,b) => a > b ? 1 : -1)
```




<br/>


### pop
Push a new element into the end of the array.

```ts
() => any

const { value, pop } = useArray([])
const lastValue = pop()
```




<br />

### shift
Removes the first element of an array, and returns that element

```ts
() => void 

const { value, shift } = useArray([])
const firstValue = shift()
```

<br />


 
### copyWithin
Copies array elements within the array, to and from specified positions

```ts
(target:number, start?:number, end?:number) => void 

const { value, pop } = useArray([])
const lastValue = pop()
```

<br/>



### filter 
Set the state to an array of elements that pass the provided test

```ts
(cb:any) => void

const { value, filter } = useArray([])
filter(v => true)
```

<br />



### fill 
Fill the elements in an array with a static value

```ts
(element:any, start?:number, end?:number) => void 

const { value, filter } = useArray([])
filter(v => true)
```

<br />



### reduce
Reduce the values of an array to a single value (going left-to-right)

```ts
(cb: any, initialValue?:any) => void

const { value, reduce } = useArray([])
reduce(v => true, 0)
```

<br />



### reduceRight
Reduce the values of an array to a single value (going right-to-left)

```ts
(cb: any, initialValue?:any) => void

const { value, reduceRight } = useArray([])
reduceRight(v => true, 0)
```

<br />



### reverse
Reverses the order of the elements in an array

```ts
() => void 

const { value, reverse } = useArray([])
reverse()
```

<br />



### sort
Sorts the elements of an array 

```ts
(cb?:any) => void 

const { value, sort } = useArray([])
sort((a,b) => a > b ? 1 : -1)
```

<br />



### splice
Adds/Removes elements from an array 

```ts
(start: number, deleteCount?: number, items?:any) => void 

const { value, splice } = useArray([])
splice(0,2, 'new')
```

<br />



### flat
Concatenate all sub-array elements into the state recursively up to the 
specified depth or 10,000 default.

```ts
(depth?: number) => void

const { value, flat } = useArray([])
flat(4)
```

<br />



### insert
Insert a new element into the array at the provided index

```ts
(index: number, element: any) => void 

const { value, insert } = useArray([])
insert(7, 'new')
```

<br />



### remove
Remove an item from the array at the provided index

```ts
(index: number) => void 

const { value, remove } = useArray([])
remove()
```

## Internal

This hook internally uses the `useState` hook to store the array and provides 
a wrapper around common array modification methods. 

Methods in `useArray` do not provide access to the current state for callback 
functions.
